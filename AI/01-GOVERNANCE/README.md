# 01-GOVERNANCE — Governance Documents

## Purpose

Index of all governance documents. Governance flows from EEC-00C (canonical) through amendments. No other governance framework is authoritative.

## Source of Truth

| Document | Location | Authority |
|----------|----------|-----------|
| EAOS.md | `../../EAOS.md` | PERMANENT — immutable |
| EEC-00C (canonical governance) | *(produced in-session, see HANDSHAKE.md)* | Ratified |
| Amendment-01 (Adoption Validation) | *(produced in-session)* | Ratified |
| **Amendment-02 (ECL-01)** | `../../EEC-00C-AMENDMENT-02-ENTERPRISE-CONTINUITY-LAYER.md` | Ratified |

## Key Rules

- All governance changes must be EEC-00C amendments (EAOS FA-04)
- Amendments can only add or clarify, never delete (EEC-00C Amendment Protocol)
- EAOS.md cannot be amended by any governance document (EAOS Ch20)

## Historical (Pre-EEC-00C)

Pre-EEC-00C governance documents (ECG series, ECG-01R series) are HISTORICAL. They do not carry governance authority but may contain useful evidence. See `PROJECT_INDEX.md` Section D and E for listing.
