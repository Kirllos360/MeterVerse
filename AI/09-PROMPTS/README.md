## Prompt Philosophy

Per EAOS.md, every task follows the 12-step lifecycle. Prompts should:
1. Reference the lifecycle step being executed
2. Reference the specific EAOS/EEC-00C rules that govern the task
3. Include the mandatory Unknown Unknown and Risk declarations
