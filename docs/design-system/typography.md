# MeterVerse Design System — Typography

## Scale
```
--text-display:  32px / 700    → Page titles
--text-heading:  24px / 600    → Section headers
--text-title:    20px / 600    → Card titles
--text-body:     16px / 400    → Body text, table cells
--text-caption:  14px / 400    → Secondary text
--text-label:    12px / 500    → Form labels, metadata
```

## Weights
```
--weight-bold:      700    → Display, emphasis
--weight-semibold:  600    → Headings
--weight-medium:    500    → Labels, active states
--weight-normal:    400    → Body text
```

## Usage by Component
| Component | Size | Weight |
|-----------|------|--------|
| Page title | display (32px) | bold |
| Section title | heading (24px) | semibold |
| Card title | title (20px) | semibold |
| Form labels | label (12px) | medium |
| Input text | caption (14px) | normal |
| Table cells | body (16px) | normal |
| Buttons | caption (14px) | medium |
| Small metadata | 10px | medium |
| Sidebar items | 12px | medium |
| Tab labels | 12px | medium |
