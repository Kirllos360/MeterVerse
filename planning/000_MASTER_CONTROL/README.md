# MeterVerse — Master Control

**Version:** 1.0  
**Status:** Active  
**Purpose:** Single source of truth for all enterprise planning, execution, and validation.

## Structure

```
000_MASTER_CONTROL/
├── Vision/          — Project vision, enterprise vision, strategic objectives
├── Rules/           — AI Bible, Definition of Done, Acceptance Criteria, Enterprise Rules
│                      Coding Rules, Review Rules, Testing Rules, Git Rules, Documentation Rules
├── Specs/           — SpecKit Master — all enterprise specifications
├── Graphs/          — Graph Master — all dependency and relationship graphs
├── Checklists/      — Master checklists for every stage
├── Status/          — PROJECT_STATUS.yaml — live project tracking
├── Dependencies/    — Master dependency map
├── Risks/           — Master risk register and mitigation
├── Timeline/        — Master timeline with milestones
├── Protocols/       — Execution, Review, Git, Documentation, Evidence, Testing protocols
└── Templates/       — Reusable templates for all planning artifacts
```
