# COMPONENT_REGISTRY.md
**Project:** MeterVerse MVEOS  
**Scope:** Reusable Visual Modules

### NAVIGATION MODULES
- `[NAV-01]` **Global Sidebar:** Multi-level hierarchy with workspace switcher.
- `[NAV-02]` **MVEOS Top Bar:** Search, Notifications, Localizer, User Profile.
- `[NAV-03]` **Metric Breadcrumb Ribbon:** System > Entity > Current navigation.

### DATA DISPLAY MODULES
- `[DATA-01]` **Smart Data Table:** High-density grid with pinned columns.
- `[DATA-02]` **Rolling KPI Card:** Metric display with sparklines and trend indicators.
- `[DATA-03]` **Infrastructure Map:** Geographic/Grid node visualization.
- `[DATA-04]` **Event Timeline:** Vertical audit log of system activities.

### WORKFLOW MODULES
- `[FLOW-01]` **Glass Action Modal:** Standard container for Create/Edit operations.
- `[FLOW-02]` **Floating Action Ribbon:** "Discard/Save" bar for unsaved states.
- `[FLOW-03]` **Wizard Stepper:** Horizontal progress indicator for multi-step flows.

### FORM MODULES
- `[FORM-01]` **Utility Input Set:** Validated fields for technical/financial data.
- `[FORM-02]` **Entity Selector:** Searchable dropdowns for Customers/Meters/Units.
- `[FORM-03]` **Advanced Filter Bar:** Expandable query builder for tables.

### FEEDBACK MODULES
- `[FEED-01]` **System Health Badge:** Pulsing status indicators (Nominal/Warning/Critical).
- `[FEED-02]` **Enterprise Toast:** Bottom-left notification with [Retry] capability.
- `[FEED-03]` **Skeleton Shell:** Layout shimmers for high-latency loading.

**Permanent Mandate:** No net-new components allowed without updating this registry.
